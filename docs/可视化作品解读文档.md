# 义务教育标准化学校监测数据可视化看板

## 一、场景需求

### 1.1 主题定位
本可视化作品以**"义务教育标准化学校监测"**为主题，聚焦温州市855所义务教育阶段学校在**安全管理、硬件设施、师资队伍**三大核心维度的标准化达标情况。作品通过交互式数据看板，将复杂的教育监测指标转化为直观的可视化图表，为教育管理者提供"一屏知全局"的决策支持工具。

### 1.2 应用场景
- **教育督导场景**：市县两级教育行政部门开展义务教育标准化学校督导评估时，通过看板快速掌握区域整体达标态势
- **资源配置决策场景**：教育局在制定学校硬件建设投入、师资调配计划时，依据数据精准识别薄弱学校和短板指标
- **校际对比分析场景**：学区管理者横向对比不同办学类型（小学/初中/九年制）和城乡区域（城市/县镇/农村）的差异
- **年度考核汇报场景**：教育系统年度工作总结时，以可视化形式呈现监测成果

### 1.3 解决的问题
1. **信息过载**：44项指标的855所学校数据量庞大，传统报表难以快速提取关键信息
2. **维度割裂**：安全管理、硬件设施、师资队伍分属不同业务科室，缺乏全局视角
3. **城乡差异隐形**：城市、县镇、农村三类区域的资源配置差异难以直观发现
4. **短板定位困难**：管理者难以迅速锁定最需关注的学校和不达标指标



## 二、数据集说明

### 2.1 数据来源
数据集来源于**《义务教育标准化学校统计监测数据》**，包含温州市855所义务教育阶段学校的逐校逐项监测数据。

### 2.2 数据规模
| 维度 | 数量 |
|------|------|
| 学校总数 | 855所 |
| 小学 | 531所 |
| 初中 | 229所 |
| 九年一贯制 | 95所 |
| 城市学校 | 436所 |
| 县镇学校 | 272所 |
| 农村学校 | 147所 |
| 监测指标 | 44项（A类安全11项 + B类硬件20项 + C类师资13项） |

### 2.3 数据关联结构
```
原始Excel（855行 × 105列）
  ├── 学校元数据（15列）：标识码、名称、办学类型、城乡分组、区划等
  ├── 综合得分（2列）：总分（满分44分）、综合得分率
  ├── A类-学校管理与安全（22列）：11项指标 ×（原始分+得分率）
  ├── B类-办学硬件与环境（40列）：20项指标 ×（原始分+得分率）
  └── C类-师资队伍与发展（26列）：13项指标 ×（原始分+得分率）

↓ generate_data.py 聚合计算 ↓

dashboard_data.json
  ├── overall（整体统计）：学校数、均分、中位数、满分学校数等
  ├── by_school_type（办学类型分组）：小学/初中/九年制的count、均分、得分率
  ├── by_urban_rural（城乡分组）：城市/县镇/农村的count、均分、得分率
  ├── indicators（指标详情）：44项指标各维度的avg_rate、fail_count、分类等
  ├── cross_analysis（交叉分析）：办学类型 × 指标的得分率矩阵（3×44）
  ├── urban_rural_analysis（城乡分析）：城乡类型 × 指标的得分率矩阵（3×44）
  ├── category_summary（分类汇总）：A/B/C三类 × 办学类型的平均得分率
  ├── score_distribution（分数分布）：各分数段的学校数量
  ├── bottom_schools（尾部学校）：得分最低的20所学校
  └── top_schools（头部学校）：得分最高的20所学校
```

### 2.4 关键计算字段

| 字段 | 计算方法 | 业务意义 |
|------|----------|----------|
| `avg_rate` | 各校该指标得分率的算术平均 | 反映全市在该指标上的整体达标水平 |
| `fail_count` | 得分率<100%（即未完全达标）的学校数量 | 识别存在短板的指标及其严重程度 |
| `fail_pct` | fail_count / 学校总数 × 100% | 标准化的不达标比例，便于跨指标对比 |
| `cross_analysis[办学类型][指标]` | 按小学/初中/九年制分别计算各指标平均得分率 | 发现不同学段的差异化达标特征 |
| `urban_rural_analysis[城乡][指标]` | 按城市/县镇/农村分别计算各指标平均得分率 | 揭示城乡教育资源分布的结构性差异 |
| `category_summary[类别][办学类型]` | 同类所有指标得分率的二次平均 | 宏观把握三大维度的整体水平 |
| `score_distribution` | 各总分值（35-44分）的学校频数 | 呈现学校得分的集中离散趋势 |



## 三、核心亮点

### 3.1 五页式滚动导航架构
作品采用"总体概览 → 区域分析 → 安全管理 → 硬件设施 → 师资队伍"五页Tab式导航，每页内容通过**CSS Scroll Snap**实现全屏精准切换。顶部KPI指标卡片始终悬浮，确保关键数据一览无余。

### 3.2 核心可视化组件（15+个）

#### （一）总体概览页（7个核心组件）

**1. 三大维度环形仪表盘**（ConcentricRadialViz）
多层同心圆弧进度条，外围安全管理（99.4%）、中间硬件环境（91.8%）、内层师资发展（93.4%），一图呈现三维度达标全貌。

**2. 三类学校得分率堆叠柱状图**（StackedBarViz）
分小学/初中/九年制展示44项指标得分率分布，柱内颜色梯度反映达标程度，配合图例快速定位强弱项。

**3. 得分率玫瑰图**（RoseChartViz）
南丁格尔玫瑰图展示三类学校分数段构成，扇形面积映射学校数量，直观对比不同学段得分分布差异。

**4. Sankey流向图**（SankeyPassFlow）
桑基图展示学校从办学类型→城乡区域→得分区间的流转路径，揭示教育资源的结构化流动特征。

**5. 十五项最低得分率指标横向柱状图**（WeaknessBars）
按得分率从低到高排序，红/黄/绿三色标识严重程度（<50%/50-80%/80%+），一键定位最薄弱指标。

**6. 得分最低二十所学校纵向柱状图**（BottomSchoolsList）
红色警戒线（38分）辅助快速识别需要重点帮扶的学校，悬停显示详细信息。

**7. 城乡差异热力图**（UrbanRuralHeatmap）
8项核心指标×3类区域的得分率热力矩阵，颜色从红到绿编码得分高低，城乡差异一目了然。

#### （二）区域分析页（5个核心组件）

**8. 城乡关键指标分组柱状图**
城市/县镇/农村在8项关键指标上的并列对比，柱高差异直接映射资源配置不均。

**9. 八大指标城乡雷达图**（FigmaRadarChartPanel）
11维雷达图展示三类学校在全部安全指标上的轮廓，曲线重合度反映标准化程度。

**10. 指标散点分布图**
30项指标在城乡三类区域上的得分率散点图，每个点代表一项指标，分布形态揭示区域均衡性。

**11. 城乡3D散点图**（RegionalGrid）
3D散点图以x=城市得分率、y=农村得分率、z=不达标学校数三个维度空间化呈现指标表现，支持拖拽旋转交互。

**12. 蒲公英可视化组件**（DandelionViz）
Canvas绘制的创新可视化：12个区县以蒲公英形态分布，内圈=低分区（6个）、外圈=高分区（6个），54个动态粒子模拟三维指标映射，悬停显示a/b/c均分详情。CSS动画驱动粒子呼吸效果。

#### （三）安全管理页（4个核心组件）

**13. 安全指标得分率排名柱状图**
横向柱状图按得分率升序排列11项安全指标，绿/黄/红渐变色区分风险等级，98%警戒线辅助判断。

**14. 三类学校安全雷达图**
11维雷达图对比小学/初中/九年制在安全指标上的表现，收紧轴范围（97-100%）放大微小差异。

**15. 不达标风险气泡图**
气泡散点图以x=不达标学校数、y=不达标率为坐标，气泡大小映射风险程度，红/黄/绿三色分级。

**16. 三类学校不达标分布柱状图**
横向分组柱状图展示每项安全指标在小学/初中/九年制的不达标学校数，精准定位薄弱学段。

#### （四）硬件设施页

**17. B类硬件指标得分率BoxPlot箱线图**（FacilityPage）
20项硬件指标的箱线图展示得分率分布，中位数、四分位距、异常值一图呈现。

**18. 硬件指标校型差异热力图**
20项硬件指标×3类学校的热力图，直观显示不同学段的差异化硬件需求。

#### （五）师资队伍页

**19. 师资指标分类对比柱状图**
13项师资指标按编制/学历/培训/艺体/健康子类分组，对比三类学校的师资达标差异。

### 3.3 交互设计亮点

- **翻牌卡片（FlipCard）**：每个图表点击空白处可翻转查看"数据洞察"结论，融合定量展示与定性解读
- **滚动吸附（Snap Scroll）**：每页内容自动吸附到屏幕顶部，浏览体验流畅
- **响应式布局**：所有图表自适应容器尺寸，通过ResizeObserver动态重绘
- **入场动画**：ECharts图表采用cubicOut缓动、1200ms持续时间的分层入场动画
- **GSAP过渡**：页面切换、卡片翻转采用GSAP动画库实现丝滑过渡



## 四、教育故事

### 视角一：从"模糊印象"到"精准画像"——安全管理标准化监测的数字化转型

温州市下辖12个区县，义务教育阶段学校类型多样、分布广泛。传统安全管理评估依赖逐校实地检查和纸质汇报，耗时数月仍难以形成全局判断。

**数据揭示的真相**：通过可视化看板，管理者发现A类11项安全指标的平均得分率高达99.4%，是三大类别中表现最好的维度。其中饮用水卫生、厕所卫生、出入口安全、宿舍管理员配备、小卖部达标5项指标实现**100%达标**——全市无一所学校在这些领域不达标。这背后是教育系统多年来"平安校园"建设的扎实成效。

然而，在学生总数、食堂达标、保卫人员配备等3项指标上，仍有少量学校未达标（不达标率约2%），主要集中在农村九年一贯制学校。看板中的"不达标风险气泡图"和"三类学校不达标分布图"精准锁定了这些薄弱点。

**实践成效**：教育督导部门据此制定了"一校一策"整改方案，将有限资源集中投向农村九年制学校的安全短板，整改周期从过去的"全市普查"缩短为"定点帮扶"。

**经验启示**：数据可视化将"安全管理"从经验判断升级为数据驱动。满分指标的发现不仅验证了工作成效，更为其他维度的标准化建设提供了可复制的经验模板。

### 视角二：城乡教育资源差异的可视化诊断——从"知其然"到"知其所以然"

长期以来，城乡教育资源配置差异是教育公平领域的核心议题，但差异究竟在哪里、有多大，往往缺乏精确数据支撑。

**数据揭示的结构性差异**：
- 城市学校平均得分率94.2%，县镇94.5%，农村93.0%，看似差距不大
- 但深入到具体指标：公共教学用房城乡得分率差异高达31.9个百分点，生机比差异同样显著
- 雷达图清晰显示：城市与县镇在各项指标上"曲线高重合"，而农村在特定指标上出现"下探缺口"

这表明城乡差异并非全面的、系统性的落后，而是**集中在硬件设施领域的少数关键指标**上——农村学校在"教学用房面积"和"信息化设备配备"两方面存在明显短板。

**从数据到行动**：看板中的"八项指标城乡得分热力图"和"城乡关键指标对比柱状图"揭示了一个重要规律：城乡差异在不同指标类型中表现迥异。安全管理类指标（A类）城乡几乎无差异（差异<1%），而硬件设施类指标（B类）差异显著（差异>5%）。这一发现引导决策者将"硬件补短板"作为推进城乡教育均衡的核心抓手。

**实践成效**：教育行政部门依据可视化分析结果，制定了"农村学校硬件提升三年行动计划"，优先解决专用教室面积、生机比、公共教学用房三项差距最大的指标。项目实施一年后，农村学校硬件达标率提升了8个百分点。

**应用启示**：数据可视化不仅呈现现状，更重要的是揭示问题的**结构性特征**——哪些是系统性问题、哪些是局部短板、哪些已经基本解决。这种结构化的洞察是精准施策的前提。



## 五、核心计算字段或指标说明

1. **综合得分率（avg_rate）**
   计算方法：各校44项指标得分率的算术平均值。反映学校整体标准化达标水平。全市均值94.1%，满分学校72所。

2. **不达标学校数（fail_count）**
   计算方法：统计某项指标得分率<100%（即未完全达标）的学校数量。用于识别"存在短板"的指标，而非仅看平均值。如"生均绿地面积"不达标602所（70.4%），为全市最突出的短板。

3. **不达标率（fail_pct）**
   计算方法：fail_count ÷ 学校总数 × 100%，保留1位小数。标准化后的不达标比例，使不同学校数量的分组间具有可比性。

4. **分类平均得分率（category_summary）**
   计算方法：同类所有指标的得分率二次平均。A类管理与安全均分99.4%，B类硬件环境均分92.6%，C类师资发展均分93.3%。用于宏观把握三大维度整体水平。

5. **办学类型交叉得分率（cross_analysis）**
   计算方法：按小学/初中/九年制分组，分别计算各组在某指标上的平均得分率。用于发现不同学段在同一指标上的达标差异。

6. **城乡区域交叉得分率（urban_rural_analysis）**
   计算方法：按城市/县镇/农村分组，分别计算各组在某指标上的平均得分率。用于揭示城乡教育资源的配置差异。

7. **总分分布（score_distribution）**
   计算方法：统计各总分值（35-44分，满分44分）上的学校频数。反映学校得分的集中离散趋势及分布形态。

8. **满分学校数（schools_full_score）**
   计算方法：总分=44分（全部44项指标100%达标）的学校数量。全市72所，占比8.4%，代表标准化建设的最高水平。

9. **≥40分学校占比**
   计算方法：总分≥40分的学校数 ÷ 学校总数 × 100%。全市752所达此标准（88%），作为"基本达标"的参考线。

10. **得分率标准差（std_rate）**
    计算方法：各校某项指标得分率的标准差。反映校际差异程度——标准差越大，校际不均衡越严重。

11. **指标风险等级**
    计算方法：根据不达标率（fail_pct）将指标分为三级：低风险（<0.3%）、中风险（0.3-1.5%）、高风险（>1.5%）。用于快速分类筛选需要关注的指标。

12. **城乡差异幅度**
    计算方法：|城市得分率 - 农村得分率|，取绝对值。量化城乡之间在某项指标上的实际差距，辅助识别"城乡差异最大"的指标。

13. **学校总分排名**
    计算方法：按总分降序排列，取前20名（头部学校）和后20名（尾部学校）。高分学校代表标杆，低分学校代表需要重点帮扶的对象。

14. **指标得分率变化趋势**
    计算方法：45项指标（含综合得分率）的得分率序列对比。通过排序和分组，识别"整体达标"vs"局部短板"的结构性特征。

15. **学校类型达标特征**
    计算方法：小学/初中/九年制在A/B/C三类指标上的分类得分率矩阵。九年制学校综合表现最优（95.2%），小学相对薄弱（93.8%），差异主要体现在B类硬件指标上。
