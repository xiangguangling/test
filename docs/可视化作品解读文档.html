义务教育标准化学校监测数据可视化看板
可视化作品解读文档 v2.0

一、场景需求

1.1 主题定位

聚焦温州市855所义务教育阶段学校在安全管理、硬件设施、师资队伍三大维度的标准化达标情况，为教育管理者提供"一屏知全局"的决策支持工具。

1.2 应用场景

  • 教育督导：快速掌握区域达标态势，精准识别薄弱学校
  • 资源配置：依据数据制定硬件投入、师资调配计划
  • 校际对比：横向对比不同办学类型和城乡区域差异
  • 年度汇报：以可视化形式呈现监测成果

二、数据集说明

2.1 数据规模

维度数量
学校总数855所（小学531 / 初中229 / 九年制95）

2.2 关键计算字段

字段计算意义
avg_rate各校得分率算术平均整体达标水平
fail_count未完全达标的学校数短板严重程度
fail_pctfail_count / 学校总数标准化不达标比例
cross_analysis办学类型 × 指标得分率矩阵学段差异
urban_rural_analysis城乡 × 指标得分率矩阵城乡差异
category_summary同类指标二次平均宏观维度水平

六、教育故事

6.1 安全管理：从"逐校检查"到"一屏掌控"

A类11项安全指标平均得分率高达99%，三图表三等分布局一屏呈现完整全景。饮用水卫生、厕所卫生等多项指标满分达标；少数不达标集中在学生总数、食堂达标等指标，主要在农村九年制学校。据此制定"一校一策"方案，从"全市普查"升级为"定点帮扶"。

6.2 城乡差异：从"感觉有差距"到"精确知差距"

城乡差异并非全面落后，而是集中在硬件设施的少数关键指标。5个图表多维揭示：城市与县镇曲线高重合，农村在公共教学用房、生机比处明显下探。推动"硬件补短板"成为城乡均衡的核心抓手。
